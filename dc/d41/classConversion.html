Detailed Description

Internal template class used to determine type convertibility.

Author
Mike Williams

Note: This class follows very closey the work of Andrei Alexandrescu from Modern C++ Design, Ch. 2, Sec. 7.

Conversion<T,U> determines whether or not a converion from T to U exists. If such a conversion does exist, then Conversion<T,U>::exists is true, otherwise it's false. If T and U are the same type, then Conversion<T,U>::same_type is true, otherwise it's false.
